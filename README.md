### Overview :
Bubble Shooter is a simple and addictive Android arcade game where players shoot colored bubbles to match and pop them. The goal is to clear all bubbles from the screen by matching three or more bubbles of the same color.


## 📱 Features

- Classic bubble shooter gameplay
- Smooth aiming and shooting controls
- Colorful bubble graphics
- Score tracking system
- Simple and user-friendly interface
- Optimized for Android devices



## 🛠 Technologies Used

- Java
- Android Studio
- Android SDK
- XML Layout Design



## 🎯 Gameplay

Players control a bubble launcher at the bottom of the screen and shoot bubbles toward a cluster above. When three or more bubbles of the same color connect, they burst and disappear.

The objective is to clear all bubbles while achieving the highest score possible.
